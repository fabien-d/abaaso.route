# abaaso.route
The routing module makes it extremely easy to put an event listener on a URI hashtag:

$.route.set("hash", function () { alert("viewing #hash!"); }); 

### del(name)
Deletes a route listener

### load(name)
Loads a route listener

### set(name, listener)
Sets a route listener
